Your key is sent directly to api.groq.com and never to any other server.

How to get a free Groq API key
  1. Go to console.groq.com
  2. Sign up or log in with your account.
  3. Open API Keys in the top right.
  4. Click Create API Key, give it a name, and copy it.
  5. Paste it above. Free tier is available — no credit card required.
